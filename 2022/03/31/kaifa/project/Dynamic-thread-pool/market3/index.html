【开发】动态线程池组件记录3 - 简历

分类于 开发 ， 项目 ， 动态线程池组件

项目名称： 动态线程池【组件】

核心技术：SpringBoot、SpringBoot Starter、ThreadPoolExecutor、Redis

项目描述：线程池的参数配置需要遵循一定的规范。但实际场景中，线程池的配置信息无法一次明确，随着业务的变更或者服务器的压力，线程池需要动态修改。因此开发了一款动态线程池组件，监控和动态修改线程池。

核心职责

  • 设计 Redis 版本的注册中心，管理线程池核心信息。定时扫描线程池，上报到 Redis 注册中心，让各个接入该组件的应用实现线程池的统一管理。
  • 开发管理系统 监控和动态修改 线程池，通过Redis 的发布订阅能力，让对应的应用上的 sdk 监听和修改线程池，实现线程池的动态修改。
